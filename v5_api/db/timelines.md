# timelines
我的动态，动态只会增加，不会减少，比如取消关注不触发任何动态


### 数据结构
动态的类型有如下几种：

|type|说明|
|:---|:---------|
|follow_user |我 关注了作家 xxx|
|follow_collection |我 关注了专题 xxx|
|support_article |我 喜欢了文章  xxx|
|comment_article |我 评论了文章  xxx|
|sign_up |我 加入了博客中国|
